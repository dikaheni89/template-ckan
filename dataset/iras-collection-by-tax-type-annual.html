Example: Javascript »

A simple ajax (JSONP) request to the data API using jQuery.

var data = {
  resource_id: '78c7c9db-c42f-4f93-b711-e8041aaa7933', // the resource id
  limit: 5, // get 5 results
  q: 'jones' // query for 'jones'
};
$.ajax({
  url: 'https://data.gov.sg/api/action/datastore_search',
  data: data,
  dataType: 'jsonp',
  success: function(data) {
    alert('Total results found: ' + data.result.total)
  }
});

About this dataset

Total amount of taxes collected by IRAS with a breakdown by tax type for Financial Year (FY)2002 to FY2021. IRAS' tax collections include Corporate Income Tax, Individual Income Tax, Withholding Tax, Goods and Services Tax, Property Tax, Stamp Duty, Betting Taxes and Estate Duty.
