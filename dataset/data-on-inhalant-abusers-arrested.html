Example: Javascript »

A simple ajax (JSONP) request to the data API using jQuery.

var data = {
  resource_id: 'ab75ed14-5a82-480a-b3c9-72e46176e036', // the resource id
  limit: 5, // get 5 results
  q: 'jones' // query for 'jones'
};
$.ajax({
  url: 'https://data.gov.sg/api/action/datastore_search',
  data: data,
  dataType: 'jsonp',
  success: function(data) {
    alert('Total results found: ' + data.result.total)
  }
});

About this dataset

The dataset gives the number of inhalant abusers arrested with a breakdown by various demographic factors eg. age group, ethnic group and gender.

Notes: The 2021 statistics are confirmed.

For more details on the Drug and Inhalant Situation, please refer to https://www.cnb.gov.sg/
